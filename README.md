# 🖥️ Gerador de Senha

Um gerador de senhas em Python com interface gráfica utilizando Tkinter. A aplicação permite ao usuário personalizar completamente a geração de senhas, escolhendo os tipos de caracteres (letras, números e símbolos) e definindo o tamanho desejado.

---

## 📦 Requisitos

* Python 3.10 ou superior
* tkinter instalado no sistema

---

## ⚙️ Instalação

### 1. Clonar o repositório

```bash
git clone https://github.com/seu-usuario/seu-projeto.git
cd seu-projeto
```

---

### 2. Criar e ativar ambiente virtual

```bash
python -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

---

## 🧩 Dependências do sistema

Este projeto utiliza **tkinter**, que pode não estar instalado por padrão em alguns sistemas.

### 🐧 Linux (Ubuntu/Debian)

```bash
sudo apt update
sudo apt install python3-tk
```

### 🐧 Fedora

```bash
sudo dnf install python3-tkinter
```

### 🪟 Windows

O tkinter já vem incluído na instalação padrão do Python.
Se não funcionar, reinstale o Python marcando a opção **"tcl/tk and IDLE"**.

---

## ▶️ Como executar

```bash
python main.py
```

---

## 💡 Observações

* Não há dependências externas (pip)
* Toda a interface é feita com tkinter
* Compatível com Windows e Linux (com tkinter instalado)
